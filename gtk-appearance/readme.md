Gtk Appearance
=======

Utility for adjusting Gtk appearance

How to install?
````
git clone https://github.com/simargl/vala-apps
cd vala-apps/gtk-appearance
make install
gtk-update-icon-cache /usr/share/icons/hicolor
glib-compile-schemas /usr/share/glib-2.0/schemas
````
Dependencies:
````
gtk+-3.0 >= 3.10
````
